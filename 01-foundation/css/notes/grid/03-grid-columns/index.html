CSS Grid Columns

This lesson continues from 02-grid-gaps . Now we will learn how to control the horizontal position of grid items using grid-column .

What is grid-column?

The grid-column property controls how a grid item is placed across columns. It can decide where an item starts, where it ends, and how many columns it should cover.

  • grid-column-start defines where the item starts.
  • grid-column-end defines where the item ends.
  • grid-column is a shorthand for start and end.
  • span means the item should cover a certain number of columns.
