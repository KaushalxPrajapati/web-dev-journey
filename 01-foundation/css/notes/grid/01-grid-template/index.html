CSS Grid Template

This is the first lesson of CSS Grid revision. Here we learn how to create rows and columns using grid-template-columns and grid-template-rows .

What is a grid template?

A grid template defines the structure of a grid. It tells the browser how many columns and rows the grid should have, and how much space each column or row should take.

  • display: grid; turns an element into a grid container.
  • Only direct children of the grid container become grid items.
  • grid-template-columns defines the columns.
  • grid-template-rows defines the rows.
  • fr means fraction of the available space.

Practice

  1. Try grid-template-columns: 100px 100px 100px; .
  2. Try grid-template-columns: repeat(3, 100px); .
  3. Try grid-template-columns: repeat(3, 1fr); .
  4. Try uneven columns: grid-template-columns: 1fr 2fr 1fr; .
  5. Add more grid items and observe how they move to the next row.
