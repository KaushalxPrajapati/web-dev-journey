CSS Grid Rows

This lesson continues from 03-grid-columns . In the previous lesson, we controlled the horizontal placement of grid items. In this lesson, we will control the vertical placement of grid items using grid-row .

What is grid-row?

The grid-row property controls how a grid item is placed across rows. It can decide where an item starts, where it ends, and how many rows it should cover.

  • grid-row-start defines where the item starts vertically.
  • grid-row-end defines where the item ends vertically.
  • grid-row is the shorthand for row start and row end.
  • span means the item should cover a certain number of rows.
