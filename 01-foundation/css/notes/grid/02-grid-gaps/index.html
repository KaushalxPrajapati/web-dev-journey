CSS Grid Gaps

This lesson continues from 01-grid-template . In the previous lesson, we created rows and columns. In this lesson, we will learn how to add spacing between those rows and columns.

What is gap?

In CSS Grid, gap is used to create space between grid items. It is better than using margin because the spacing is controlled by the grid container, not by each individual item.

  • row-gap adds space between rows.
  • column-gap adds space between columns.
  • gap is a shorthand for both row and column gaps.
